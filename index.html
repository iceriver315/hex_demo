Hexadecimal

The Hexadecimal, commonly referred to as " hex," is a numerical system that uses 16 symbols to represent numbers. It is often used as a shorthand notation for binary numbers in computing and programming.

In hex, the symbols 0-9 represent the decimal numbers 0-9, and the symbols A-F represent the decimal numbers 10-15.

Hexadecimal is commonly used to represent memory addresses, color codes, and binary data in a more concise and readable format.

ASCII Text

ASCII(American Standard Code for Information Inter change) is a character encoding standard that represents characters as numeric codes. It has become the most widely used character encoding scheme.

ASCII uses a 7-bit code to represent 128 characters, including uppercase and lowercase letters, numbers, punctuation marks, and control characters.In the ASCII code, each of these characters are assigned a decimal number from 0 to 127. For example, the ASCII representation of upper case A is 65 and the lower case a is 97.

It provides a standardized way for computers to interpret and display text, allowing for compatibility and communication between different systems and programming languages.
